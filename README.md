# Neuro-Symbolic Vision Language Framework for Explainable Medical Diagnosis

## 📌 Overview
This project presents a **Neuro-Symbolic AI framework** for medical diagnosis that combines **deep learning** with **symbolic reasoning** to deliver **accurate and interpretable predictions**.

The system integrates:
- Convolutional Neural Networks (CNN) for medical imaging  
- ClinicalBERT for clinical text understanding  
- Multimodal feature fusion  
- A symbolic reasoning layer for interpretability  
- Explainable AI techniques for transparent decision-making  

---

## 🎯 Objective
To develop a **clinically reliable and explainable diagnostic system** that overcomes the limitations of black-box AI models by providing **interpretable insights alongside predictions**.

---

## 🏗️ Methodology
1. **Image Processing** → CNN extracts features from chest X-rays  
2. **Text Processing** → ClinicalBERT encodes clinical notes  
3. **Feature Fusion** → Combines multimodal representations  
4. **Reasoning Layer** → Applies symbolic logic for decision support  
5. **Explainability** → Generates visual and feature-based explanations
